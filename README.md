# Chat en Tiempo Real 

# Instalacón
```
git clone https://github.com/GrupoWeb/redesProyecto.git NombreProyecto
cd NombreProyecto
composer install
npm install
php artisan key:generate
cp .env.example
crear la base de datos y configurar el archvio .env con las credenciales de la conexión
(dentro del archivo .env.example se encuentra la configuración de boardcast 
que se utilizar para la parte segura)
```
